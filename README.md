# CareerPilot AI 🚀

An AI-powered career guidance platform built using Java Full Stack and Large Language Models (LLMs).

## Tech Stack

### Frontend

* React.js
* Vite
* HTML5
* CSS3
* JavaScript

### Backend (Planned)

* Java
* Spring Boot
* REST APIs
* Spring Security

* MySQL

* OpenAI API / Gemini API
* Prompt Engineering
