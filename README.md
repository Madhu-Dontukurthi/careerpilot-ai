# CareerPilot AI 🚀

An AI-powered career guidance platform built using Java Full Stack and Large Language Models (LLMs).

## Tech Stack

* Frontend: React.js, Vite, HTML, CSS, JavaScript
* Backend: Java, Spring Boot, REST APIs
* Database: MySQL
* AI Integration: OpenAI API / Gemini API

## Features (Planned)

* User Authentication
* AI Career Guidance
* Resume Analysis
* Skill Gap Analysis
* Learning Roadmap Generator
* Progress Tracking Dashboard
